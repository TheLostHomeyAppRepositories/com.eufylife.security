Инструкции по установке

Из-за особенностей работы закрытого API Eufy Security, комбинация email/пароль не может использоваться одновременно в мобильном приложении Eufy Security и приложении Homey.

Рекомендуется использовать мобильное приложение Eufy для создания вторичного "администраторского" аккаунта с отдельным адресом электронной почты и использовать его с этим приложением.

Использование
- Установите это приложение на ваш Homey.
- Перейдите к `добавить устройства`
- Введите ваше Имя пользователя и Пароль. Нажмите Далее
- Доступные устройства будут показаны
