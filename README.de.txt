Installationsanweisungen

Aufgrund der Funktionsweise der privaten API von Eufy Security kann eine Kombination aus E-Mail und Passwort 
nicht sowohl mit der mobilen Eufy Security-App als auch mit der Homey-App verwendet werden.

Es wird empfohlen, die mobile Eufy-App zu verwenden, um ein sekundäres "Admin"-Konto mit einer separaten E-Mail-Adresse zu erstellen
und dieses mit dieser App zu verwenden.

Verwendung
- Installiere diese App auf deinem Homey.
- Gehe zu „Geräte hinzufügen“
- Gib deinen Benutzernamen und dein Passwort ein. Klicke auf „Weiter“
- Verfügbare Geräte werden angezeigt
