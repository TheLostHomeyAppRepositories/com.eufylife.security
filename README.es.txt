Instrucciones de instalación

Debido a la forma en que funciona la API privada de Eufy Security, una combinación de correo electrónico/contraseña no puede funcionar tanto con la aplicación móvil de Eufy Security como con la aplicación de Homey.

Se recomienda usar la aplicación móvil de Eufy para crear una cuenta "admin" secundaria con una dirección de correo electrónico diferente y usarla con esta aplicación.

Uso
- Instala esta aplicación en tu Homey.
- Ve a `añadir dispositivos`
- Proporciona tu Nombre de Usuario y Contraseña. Haz clic en Siguiente
- Los dispositivos disponibles aparecerán
