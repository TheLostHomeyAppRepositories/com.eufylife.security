Installationsinstruktioner

På grund av hur Eufy Securitys privata API fungerar kan en kombination av e-post/lösenord inte fungera med både Eufy Securitys mobilapp och Homey-appen.

Det rekommenderas att använda Eufys mobilapp för att skapa ett sekundärt "admin" konto med en separat e-postadress och använda det med den här appen.

Användning
- Installera den här appen på din Homey.
- Gå till `lägg till enheter`
- Ange ditt användarnamn och lösenord. Klicka på nästa
- Tillgängliga enheter visas
