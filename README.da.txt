Installationsinstruktioner

På grund af måden Eufy Securitys private API fungerer på, kan en e-mail/adgangskode kombination ikke bruges med både Eufy Security mobilappen og Homey-appen. 

Det anbefales at bruge mobilappen Eufy til at oprette en sekundær "admin"-konto med en separat e-mailadresse og bruge den med denne app.

Brug
- Installer denne app på din Homey.
- Gå til `tilføj enheder`
- Indtast dit Brugernavn og Adgangskode. Klik på Næste
- Tilgængelige enheder vises
