Installering instruksjoner 

På grunn av måten Eufy Security private API fungerer, kan en e-post/passord kombinasjon ikke fungere både med Eufy Security mobilapp og Homey appen. 

Det anbefales å bruke mobilappen til Eufy for å lage en sekundær "admin" konto med en separat e-postadresse og bruke den med denne appen.

Bruk
- Installer denne appen på din Homey.
- Gå til `legg til enheter`
- Oppgi ditt brukernavn og passord. Klikk Neste
- Tilgjengelige enheter vises
